# Mastoline

![logo](img/logo.png)

Mastodon on your terminal! 

Mastoline is a simple mastodon cilent that runs in the terminal.

It has support for

* Viewing statuses
* Replying
* Tooting
* Rich text
* [+] other things I am too lazy to mention
